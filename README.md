# SQL-Business-Intelligence-Revenue-Analysis

Project Overview:
This project involves solving 23 complex business questions using Advanced SQL. The goal was to transform raw transactional data into actionable insights for a retail environment.

Key Technical Challenges Solved:
* Growth Metrics: Calculated monthly sales growth rates and customer acquisition trends.
* Database Objects: Developed Stored Procedures for product sales and created Views to summarize customer payment history.
* Advanced Logic: Identified high-value orders containing products from multiple categories.

Tools Used: PostgreSQL / MySQL.
